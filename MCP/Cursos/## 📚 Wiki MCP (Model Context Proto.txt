## 🧩 **¿Qué es MCP?**

![image.png](attachment:2912a03f-4260-4464-9c8b-12d5441d3984:image.png)

**Definición:**

> El Model Context Protocol (MCP) es un estándar abierto creado por Anthropic en 2024, que permite conectar fácilmente modelos de lenguaje (como ChatGPT o Claude) con datos externos y herramientas en tiempo real.
> 

**Problema que resuelve:**

- **Evita las integraciones fragmentadas y repetitivas.**
- **Proporciona un estándar universal para conectar modelos de IA con múltiples fuentes de información.**

---

## ⚙️ **¿Cómo funciona MCP?**

[Aadit Sheth on Twitter / X](https://x.com/aaditsh/status/1898216925580476590)

![image.png](attachment:1ee073b3-d820-4940-9fdc-748977a5e071:image.png)

![image.png](attachment:14cfef6c-e1e8-449b-aebf-1e0503da0c68:image.png)

**Modelo Cliente-Servidor:**

- Cliente (modelo IA): aplicación que utiliza un LLM (por ejemplo, ClaudeDesktop).
- Servidor MCP: herramienta o interfaz que conecta una fuente externa (Google Drive, Bases de datos, APIs, etc.) con el modelo de IA.

**Flujo básico (paso a paso):**

1. El cliente (IA) inicia una sesión con el servidor MCP.
2. Solicita información o acciones mediante el protocolo.
3. El servidor ejecuta la solicitud sobre el sistema externo.
4. Retorna los resultados en formato JSON estructurado.
5. El modelo de IA usa estos resultados para generar respuestas más precisas.

---

## 🚀 **Ventajas clave de MCP:**

- Estandarización (similar a LSP en desarrollo de software).
- Escalabilidad en integraciones de IA.
- Seguridad controlada en el acceso a datos.
- Mayor calidad y contexto en respuestas generadas por IA.

## 🔍 **Aplicaciones prácticas con ejemplos**

| Caso de uso | Ejemplo práctico |
| --- | --- |
| 📁 **Gestión documental** | Consultar documentos actualizados desde Google Drive o Box para responder preguntas internas en tiempo real. |
| 💻 **Asistente de código** | Permitir a un asistente como Claude acceder a documentación del proyecto, historial de commits o información específica desde GitHub para ayudar a programar con contexto actualizado. |
| 📊 **Consultas de bases de datos** | Obtener información en tiempo real desde bases de datos para generar respuestas precisas sobre inventarios, ventas, o estadísticas del negocio. |
| 🔧 **Automatización de tareas** | Crear tickets automáticamente en Jira o enviar correos electrónicos desde el modelo tras solicitudes del usuario usando funciones expuestas por MCP. |
